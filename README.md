#  Caesar Cipher & Brute-Forcer

A simple Python script for encrypting and decrypting messages using the Caesar Cipher. It includes a built-in brute-force tool to crack messages without a key.

##  Features
- **Encrypt/Decrypt:** Shift characters with any integer key.
- **Brute-Force:** Instantly prints all 25 possible decryptions.
- **Safe:** Preserves spaces and special characters.

## What I learned
- Character encoding with ord() and chr().
- Circular shifts with modulo % 26.
- Handling user input and loops in Python.
